## 建议学习路线

```
第1周: 完成全部 ★☆☆☆☆ 题目 (2020-2023 的 T1 题，共12道)
        → 掌握基本输入输出、数组操作、简单数学计算

第2周: 完成 ★★☆☆☆ 题目 (T2 题，共6道)
        → 掌握前缀和、差分、排序、基本模拟

第3周: 挑战 ★★★☆☆ 题目 (较难的 T2/T3，共4道)
        → 掌握较复杂的算法思维、优化技巧

第4周: 复习 + 限时模拟
        → 4小时内完成5道题的模拟考试
```

---

## 考试技巧

1. **T1 必拿分**：第一题通常极简单，10分钟内完成
2. **T2 争取拿分**：第二题中等难度，30-45分钟完成
3. **T3 大模拟**：细心读题，画流程图，60分钟
4. **T4/T5 部分分**：即使不会正解，也要写暴力拿部分分
5. **先通读全部题目**，从简单开始做

## 编译与运行

所有题解使用标准 C11，编译命令：

```bash
# 一般题目（不需要数学库）
gcc -std=c11 -O2 -Wall 题目.c -o 题目
./题目 < 输入文件

# 使用 math.h 的题目（202206-1, 202212-1, 202309-1, 202312-2等）
gcc -std=c11 -O2 -Wall 题目.c -o 题目 -lm
./题目 < 输入文件
```
