## 202203-1 未初始化警告

### 问题描述
一个程序有 k 条语句，每条语句的格式为：

```
x_i = y_i
```

其中 x_i 是左值（被赋值的变量），y_i 是右值（被引用的变量）。

如果右值变量 y_i 在当前语句之前**没有被赋值过**（即未初始化），则编译器会给出一个"未初始化警告"。

注意：右值 y_i 为 0 时表示一个常量 0，不会引起警告。

给定 k 条语句，求编译器会给出多少个"未初始化警告"。

### 输入格式
第一行包含两个整数 n 和 k，表示变量的数量和语句的数量（变量编号为 1~n）。

接下来 k 行，每行包含两个整数 x_i 和 y_i，表示一条赋值语句。

### 输出格式
输出一个整数，表示未初始化警告的数量。

### 样例输入
```
10 7
1 2
2 3
3 4
4 2
3 5
4 6
5 6
```

### 样例输出
```
2
```

### 样例解释
- 第1条：y=2 未初始化，警告
- 第2条：y=3 未初始化，警告
- 此后各变量已初始化，无警告

### 评测用例规模
- 1 ≤ n, k ≤ 10^5
- 1 ≤ x_i ≤ n, 0 ≤ y_i ≤ n

### 考点
数组标记、模拟

## 202203-2 出行计划

### 问题描述
最近需要出行，需要做核酸检测。在 t 时刻做的核酸检测，在 t+k 时刻生效，在 t+k+23 时刻失效（以 24 小时为有效期）。

有 n 个出行计划，第 i 个出行计划在 t_i 时刻进入某场所，该场所要求持有 c_i 小时内的核酸检测结果。

给定 m 个查询，每个查询给出一个做核酸的时刻 q，求在该时刻做核酸能满足多少个出行计划的要求。

### 输入格式
第一行包含三个整数 n, m, k。

接下来 n 行，每行两个整数 t_i 和 c_i。

接下来 m 行，每行一个整数 q，表示一个查询。

### 输出格式
输出 m 行，每行一个整数，表示对应查询的答案。

### 样例输入
```
6 2 10
5 24
10 24
11 24
34 24
35 24
35 48
1
2
```

### 样例输出
```
3
3
```

### 评测用例规模
- 1 ≤ n, m ≤ 10^5
- 1 ≤ t_i, c_i, q ≤ 2 × 10^5

### 考点
差分数组、区间计数

## 202206-1 归一化处理

### 问题描述
给定 n 个整数 a_1, a_2, ..., a_n。求归一化后的值：

$$\bar{a} = \frac{1}{n}\sum_{i=1}^{n} a_i$$

$$D(a) = \frac{1}{n}\sum_{i=1}^{n} (a_i - \bar{a})^2$$

$$f(a_i) = \frac{a_i - \bar{a}}{\sqrt{D(a)}}$$

对于每个 a_i，输出 f(a_i)。保证 D(a) > 0。

### 输入格式
第一行包含一个正整数 n。

第二行包含 n 个整数 a_1, a_2, ..., a_n。

### 输出格式
输出 n 行，每行一个浮点数 f(a_i)，保留 10 位小数。

### 样例输入
```
3
1 2 3
```

### 样例输出
```
-1.2247448714
0.0000000000
1.2247448714
```

### 评测用例规模
- 1 ≤ n ≤ 1000
- |a_i| ≤ 1000

### 考点
数学、浮点数计算、统计

## 202206-2 寻宝！大冒险！

### 问题描述
在二维平面上，绿化图可以表示为一个无穷大的平面，其中有 n 棵树，每棵树位于整数坐标。

藏宝图是一个 S×S 的 0/1 矩阵 B（下标从 0 到 S），表示在藏宝图范围内每个位置是否应该有树（1=有树，0=无树）。

问：有多少个地图上的位置 (x, y) 可以作为藏宝图的左下角，使得藏宝图 B 与绿化图在矩形 [x, x+S] × [y, y+S] 区域内**完全匹配**？

### 输入格式
第一行包含三个整数 n, L, S。

接下来 n 行，每行两个整数 x_i, y_i，表示一棵树的位置。

接下来 S+1 行，每行 S+1 个整数，表示藏宝图 B（注意输入顺序是**从下到上**的）。

### 输出格式
输出一个整数，表示匹配位置的数量。

### 样例输入
```
5 100 2
0 0
1 1
2 2
3 3
4 4
0 0 1
0 1 0
1 0 0
```

### 样例输出
```
3
```

### 评测用例规模
- 1 ≤ n ≤ 1000（树的数量）
- S ≤ 50
- 坐标范围 [0, L]，L ≤ 10^9

### 考点
枚举、哈希/查找、坐标离散化

## 202209-1 如此编码

### 问题描述
某次测验后，老师在评卷。共有 m 个得分点，第 i 个得分点的满分为 c_i。老师定义了一个奇怪的算出总成绩的方式：

对每个得分点设置权重 a_i，然后总成绩为：

$$\text{sum} = \sum_{i=1}^{m} c_{i-1} \times a_i$$

其中 c_0 = 1。

已知答案数组 b[1..m]（即正确答案），现求 a[1..m]。

### 输入格式
第一行包含两个整数 n 和 m。

第二行包含 m 个整数 c_1, c_2, ..., c_m。

### 输出格式
输出一行 m 个整数，表示 a_1, a_2, ..., a_m。

### 样例输入
```
10 3
3 5 2
```

### 样例输出
```
3 1 0
```

### 评测用例规模
- 1 ≤ m ≤ 20
- c_i ≤ 100

### 考点
进制分解、数学

## 202212-1 现值计算

### 问题描述
某项目在未来 n 年内的预期收入为：第 0 年收入 a_0，第 1 年收入 a_1，...，第 n 年收入 a_n。银行年利率为 i%。

定义第 k 年的收入折算到当前的"现值"为：

$$PV_k = \frac{a_k}{(1 + i\%)^k}$$

求该项目所有年份收入的现值之和，结果保留 3 位小数。

### 输入格式
第一行包含两个整数 n 和 i。

第二行包含 n+1 个整数 a_0, a_1, ..., a_n。

### 输出格式
输出一个浮点数，表示现值之和，保留 3 位小数。

### 样例输入
```
2 5
-1000 500 600
```

### 样例输出
```
20.408
```

### 评测用例规模
- 1 ≤ n ≤ 50
- 0 ≤ i ≤ 100
- |a_k| ≤ 10000

### 考点
数学计算、浮点数、循环
