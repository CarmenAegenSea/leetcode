## 202009-1 称检测点查询

### 题目背景
2020年6月8日，国务院联防联控机制发布《关于加快推进新冠病毒核酸检测的实施意见》，提出对密切接触者等8类重点人群"应检尽检"。

### 问题描述
某市设有 n 个核酸检测点，编号从 1 到 n。第 i 号检测点的位置可以表示为一个平面坐标 (x_i, y_i)。

市民到达检测点 i 的距离（为方便计算，采用**欧几里得距离的平方**）为：

$$D_i = (X - x_i)^2 + (Y - y_i)^2$$

其中 (X, Y) 是市民所在位置。现给定市民位置和所有检测点位置，请找出距离市民**最近**的 3 个检测点（编号）。

### 输入格式
输入共 n+1 行。

第一行包含三个整数 n, X, Y，分别表示检测点总数和市民位置坐标。

第二行到第 n+1 行，每行两个整数 x_i, y_i，表示第 i 号检测点位置。

### 输出格式
输出共三行，按距离从近到远依次输出距离市民最近的 3 个检测点编号。

如果距离相同，编号小的优先。

### 样例输入
```
5 0 0
-2 1
-1 3
1 3
3 1
0 4
```

### 样例输出
```
1
4
2
```

### 评测用例规模
- 3 ≤ n ≤ 200（简单）
- 全部坐标绝对值 ≤ 1000

### 考点
结构体排序、简单数学计算

## 202009-2 风险人群筛查

### 题目背景
某地疫情爆发后，防疫部门希望通知经过了高危区域的人员。

### 问题描述
高危区域可以用一个矩形区域表示，其左下角坐标为 (x_l, y_d)，右上角坐标为 (x_r, y_u)。

某位居民的位置记录包含 t 个时刻的坐标 (x_1, y_1), (x_2, y_2), ..., (x_t, y_t)。

- 如果某个时刻的坐标在矩形内（含边界），则该时刻位于高危区域。
- 如果某居民**连续** k 个或更多时刻位于高危区域，则认为该居民**逗留**。
- 如果某居民至少有一个时刻位于高危区域，则认为该居民**经过**。

给定 n 位居民的位置记录，统计**经过**高危区域的人数 p 和**逗留**高危区域的人数 q。

### 输入格式
输入共 n+1 行。

第一行包含七个整数 n, k, t, x_l, y_d, x_r, y_u。

接下来 n 行，每行包含 2t 个整数，表示一位居民按时间顺序的 t 个坐标。

### 输出格式
输出共两行，每行一个整数，分别表示经过高危区域的人数 p 和逗留高危区域的人数 q。

### 样例输入
```
5 2 6 20 40 100 80
10 10 30 40 50 60 70 80 90 100 90 60
10 10 30 40 50 60 70 80 90 100 120 50
10 10 30 40 50 60 70 80 100 50 30 20
10 10 30 40 50 60 70 80 90 100 110 80
10 10 30 40 50 60 70 80 90 100 110 90
```

### 样例输出
```
4
2
```

### 评测用例规模
- 1 ≤ n ≤ 20, 1 ≤ k ≤ t ≤ 1000
- 坐标绝对值 ≤ 10^6

### 考点
简单模拟、区间判断、连续段计数

## 202012-1 期末预测之安全指数

### 问题描述
安全指数 y 定义为：

$$y = \text{ReLU}\left(\sum_{i=1}^{n} w_i \cdot \text{score}_i\right)$$

其中 ReLU(x) = max(0, x)。

给定 n 个指标，每个指标包括权重 w_i 和得分 score_i，求安全指数 y。

### 输入格式
第一行包含一个正整数 n。

接下来 n 行，每行包含两个整数 w_i 和 score_i。

### 输出格式
输出一个整数，表示安全指数 y。

### 样例输入
```
6
2 60
3 50
6 80
1 60
3 50
5 80
```

### 样例输出
```
1420
```

### 评测用例规模
- 1 ≤ n ≤ 10^5
- w_i, score_i ≤ 100

### 考点
简单计算、循环累加

## 202012-2 期末预测之最佳阈值

### 问题描述
有 m 位同学，每位同学有一个安全指数 y_i 和实际是否挂科的结果 result_i（0=未挂科, 1=挂科）。

现在要选取一个阈值 θ，预测规则为：
- 如果 y ≥ θ，预测为挂科（1）
- 如果 y < θ，预测为未挂科（0）

预测正确的次数越多，θ 越好。如果有多个 θ 使得正确次数相同，选择**最大的** θ。

求最佳阈值 θ。

### 输入格式
第一行包含一个正整数 m。

接下来 m 行，每行两个整数 y_i 和 result_i。

### 输出格式
输出一个整数，表示最佳阈值 θ。

### 样例输入
```
6
0 0
1 0
1 1
3 1
5 1
7 1
```

### 样例输出
```
3
```

### 评测用例规模
- 1 ≤ m ≤ 10^5
- 0 ≤ y_i ≤ 10^8

### 考点
排序、前缀和、思维
