## 202303-1 田地丈量

### 问题描述
西西艾弗岛是一个矩形区域，左下角坐标为 (0, 0)，右上角坐标为 (a, b)。

岛上开垦了 n 块田地，每块田地是边与坐标轴平行的矩形。第 i 块田地的左下角坐标为 (x1_i, y1_i)，右上角坐标为 (x2_i, y2_i)。

这 n 块田地可能有重叠。求所有田地在岛内（即矩形 [0, a] × [0, b]）的**总面积**（重叠部分只算一次……不对，此题求的是**被开垦的田地覆盖的总面积**，重叠部分不计入多次，但实际上是求每块田与岛屿的交集面积之和，不需要去重——每块田地可能重叠，但分别属于不同农民，各算各的）。

**简化题意**：给定 n 个矩形（田地）和一个边界矩形（岛屿），对每块田地求它与岛屿的交集面积，然后将 n 个交集面积相加输出。

### 输入格式
第一行包含三个整数 n, a, b。

接下来 n 行，每行四个整数 x1_i, y1_i, x2_i, y2_i。

### 输出格式
输出一个整数，表示总面积。

### 样例输入
```
3 10 10
0 0 5 5
5 -2 15 3
8 8 15 15
```

### 样例输出
```
44
```

### 评测用例规模
- 1 ≤ n ≤ 100
- 0 < a, b ≤ 10^4
- 坐标绝对值 ≤ 10^4

### 考点
几何、矩形交集、简单枚举

## 202305-1 重复局面

### 问题描述
国际象棋的棋盘局面可以用 8 行每行 8 个字符的字符串表示。

给定 n 个局面，对于每个局面，判断它是第几次出现（包括本次）。

### 输入格式
第一行包含一个正整数 n。

接下来 8n 行，每 8 行为一个局面，每行 8 个字符。

### 输出格式
输出 n 行，每行一个整数，表示该局面是第几次出现。

### 样例输入
```
3
rnbqkbnr
pppppppp
........
........
........
........
PPPPPPPP
RNBQKBNR
rnbqkbnr
pppppppp
........
........
........
........
PPPPPPPP
RNBQKBNR
rnbqkbnr
pppppppp
........
........
........
........
PPPPPPPP
RNBQKBNR
```

### 样例输出
```
1
2
3
```

### 评测用例规模
- 1 ≤ n ≤ 100

### 考点
字符串比较、哈希、计数

## 202309-1 坐标变换（其一）

### 问题描述
对平面上的坐标点进行一系列操作。共有两种操作：

1. **平移**：将点 (x, y) 平移 (dx, dy)，变为 (x+dx, y+dy)
2. **旋转**：将点绕原点逆时针旋转 θ 度

给定 n 个操作和 m 个初始点坐标，对每个初始点，依次执行完所有 n 个操作后，输出最终坐标。

### 输入格式
第一行包含两个整数 n 和 m。

接下来 n 行，每行描述一个操作：
- 如果是平移：`1 dx dy`
- 如果是旋转：`2 θ`

接下来 m 行，每行两个整数 x 和 y，表示初始坐标。

### 输出格式
输出 m 行，每行两个浮点数，表示最终坐标，保留 4 位小数。

### 样例输入
```
3 2
1 2 3
2 90
1 -1 2
1 0
2 0
```

### 样例输出
```
2.0000 5.0000
3.0000 4.0000
```

### 评测用例规模
- 1 ≤ n, m ≤ 10^5

### 考点
数学、矩阵变换、合并操作

## 202312-1 仓库规划

### 问题描述
西西艾弗岛上有 n 个仓库，每个仓库的位置可以用 m 维空间中的坐标表示。第 i 个仓库的坐标为 (x_{i,1}, x_{i,2}, ..., x_{i,m})。

对于第 i 个仓库，如果存在另一个仓库 j，使得对于所有维度 k，都有 x_{j,k} > x_{i,k}，则称仓库 j 是仓库 i 的"上级仓库"。

如果有多个上级仓库，选择**编号最小**的那个作为仓库 i 的上级。

对于每个仓库，输出其上级仓库的编号（如果没有上级仓库，输出 0）。

### 输入格式
第一行包含两个整数 n 和 m。

接下来 n 行，每行 m 个整数，表示第 i 个仓库的 m 维坐标。

### 输出格式
输出 n 行，第 i 行输出仓库 i 的上级仓库编号。

### 样例输入
```
4 2
0 0
-1 -1
1 2
0 -1
```

### 样例输出
```
3
1
0
3
```

### 评测用例规模
- 1 ≤ n ≤ 1000
- 1 ≤ m ≤ 10
- 坐标绝对值 ≤ 10^6

### 考点
枚举、多维比较

## 202312-2 因子化简

### 问题描述
给定 q 个正整数 n，对于每个 n，将其分解为质因数的乘积：

$$n = p_1^{e_1} \times p_2^{e_2} \times ... \times p_r^{e_r}$$

现在要将 n 化简：保留所有**指数 ≥ k** 的质因数，丢掉指数 < k 的质因数（即设其指数为 0）。

化简后得到的新数字为：

$$n' = \prod_{e_i \ge k} p_i^{e_i}$$

对于每个查询，输出化简后的 n'。

### 输入格式
第一行包含一个正整数 q。

接下来 q 行，每行两个正整数 n 和 k。

### 输出格式
输出 q 行，每行一个整数，表示化简后的结果。

### 样例输入
```
3
2155895064 3
2 2
10000000000 5
```

### 样例输出
```
2238728
1
10000000000
```

### 评测用例规模
- 1 ≤ q ≤ 10
- 2 ≤ n ≤ 10^10
- 1 ≤ k ≤ 50

### 考点
质因数分解、试除法、快速幂
