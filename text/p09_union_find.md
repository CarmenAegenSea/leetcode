# 题目9：并查集（连通分量）

有 N 个人，编号 1~N。M 条关系，每条关系 `a b` 表示 a 和 b 是朋友（朋友关系可传递）。Q 次询问，每次 `a b` 问两人是否是朋友。

**输入**
第一行 N, M, Q（1 ≤ N ≤ 10^5, 0 ≤ M,Q ≤ 10^5）
接下来 M 行，每行两个整数 a b
接下来 Q 行，每行两个整数 a b

**输出**
每次询问输出 `yes` 或 `no`。

**样例**
```
5 3 4
1 2
2 3
4 5
1 3
1 4
2 5
3 5
```
```
yes
no
no
no
```

**要求：** 并查集（路径压缩+按秩合并），文件名 `union_find.c`
